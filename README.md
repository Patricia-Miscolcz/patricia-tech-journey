# 👩‍💻 Sobre o Projeto

O **Tech Journey** é meu portfólio profissional desenvolvido para apresentar minha trajetória acadêmica, experiência em tecnologia, projetos, competências e evolução na área de desenvolvimento de software.

Este projeto representa minha jornada da Matemática para a Tecnologia e demonstra como o aprendizado contínuo pode transformar carreiras.

---

# 👩‍🏫 Sobre Mim

Sou **Professora Universitária • Tutora Acadêmica • Desenvolvedora**.

Minha trajetória começou na graduação em **Matemática pela Universidade de São Paulo (USP)**, um desafio que exigiu anos de dedicação para conquistar uma vaga na FUVEST e concluir uma das formações mais respeitadas do país.

Durante a pandemia decidi iniciar uma nova jornada na tecnologia, cursando **Análise e Desenvolvimento de Sistemas pela UniFECAF**, seguida por uma **Pós-Graduação em Desenvolvimento Full Stack**.

Hoje uno minhas duas maiores paixões:

- 🎓 Educação
- 💻 Tecnologia

Meu objetivo é mostrar aos meus alunos que **nunca é tarde para aprender, recomeçar e transformar a própria história.**
